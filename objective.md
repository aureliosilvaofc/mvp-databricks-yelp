# Objetivo do MVP

## Problema
Entender padrões relacionados às avaliações de restaurantes presentes no Yelp.

## Objetivo
Construir um pipeline de dados em nuvem utilizando Databricks Community Edition para ingestão, transformação e análise de dados.

## Perguntas de negócio
1. Quais cidades possuem mais restaurantes?
2. Quais restaurantes possuem melhores avaliações?
3. Qual a média de estrelas por cidade?
4. Quais usuários são mais ativos?
5. Como está distribuída a quantidade de avaliações?

## Hipóteses
- Grandes cidades possuem maior quantidade de restaurantes.
- Restaurantes com mais avaliações tendem a possuir notas mais consistentes.
- Usuários muito ativos influenciam fortemente o volume de reviews.
