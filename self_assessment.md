# Autoavaliação

## Objetivos atingidos
Foi possível construir um pipeline de dados completo utilizando Databricks Community Edition.

As etapas implementadas incluíram:
- ingestão de dados
- transformação
- persistência
- análise exploratória
- respostas às perguntas de negócio

## Dificuldades encontradas
- Limitações de recursos do Databricks Community Edition.
- Tempo de processamento do arquivo de reviews.
- Ajustes nos paths dos arquivos no volume do Databricks.

## Trabalhos futuros
- Construção de dashboards.
- Inclusão de novos datasets.
- Implementação de análises mais avançadas.
- Criação de pipelines automatizados.

## Considerações finais
O MVP atingiu os objetivos principais propostos inicialmente e demonstrou a construção de um pipeline de dados em nuvem funcional.
